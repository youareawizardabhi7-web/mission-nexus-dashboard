 MISSION NEXUS

> The Operating System for Space Exploration

Mission Nexus is a futuristic Space Mission Control Dashboard designed for a frontend hackathon. Set in the year "2189", the project envisions a world where humanity has expanded across the Solar System, and Mission Nexus serves as the central command platform for monitoring spacecraft, autonomous fleets, astronauts, satellite networks, and emergency operations.

---

> About the Project

Mission Nexus is more than a dashboard—it's an immersive mission control experience inspired by NASA Mission Control, SpaceX, and modern sci-fi interfaces.

The project combines cinematic visuals with a clean, futuristic UI to simulate a real-time command center for deep-space exploration.

> 🛰️ Current Mission
Operation Eclipse

A powerful electromagnetic anomaly near Saturn has disrupted communications across the Nova Prime orbital colony. Mission Nexus has been activated to:

- Restore communication networks
- Coordinate autonomous rescue drones
- Monitor spacecraft telemetry
- Track mission-critical systems
- Ensure crew safety

## 🚀 Getting Started

### Clone the repository

```bash
https://github.com/youareawizardabhi7-web/mission-nexus-dashboard.git
```

### Navigate to the project

```bash
cd mission-nexus
```

### Install dependencies

```bash
npm install
```

### Start the development server

```bash
npm run dev
```
